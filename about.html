System Architecture

How The Data Flows

Step-by-step explanation suitable for students and expo judges

üåç

Step 1: Sensors Measure the Soil

Five types of sensors are inserted into the farm soil. The moisture sensor checks how wet the soil is, the pH sensor measures acidity, the NPK sensor checks nutrient levels (like fertilizer), and the DHT22 sensor reads air temperature and humidity near the field.

Every 5 seconds
üîå

Step 2: ESP32 Reads & Packages Data

The ESP32 microcontroller reads all sensor signals ‚Äî both analog (moisture, pH) and digital (NPK via RS485, DHT22). It converts raw voltage values into meaningful numbers and packages them into a JSON format like: {"moisture":72,"ph":6.8,"n":55,"p":38,"k":45}.

ESP32 Processing
üì°

Step 3: Wi-Fi Transmission

The ESP32 connects to your farm's Wi-Fi network and sends the JSON data to a cloud server using HTTP POST request or MQTT protocol. This happens wirelessly ‚Äî no wires needed between the field and the internet!

// ESP32 Arduino Code (simplified) WiFi.begin("FarmNetwork", "password"); HTTPClient http; http.begin("https://api.soilsense.io/data"); http.POST(jsonPayload); // Send to cloud

Step 4: Cloud Storage & Processing

The cloud server (Node.js + MongoDB or Firebase) receives the reading, stamps it with a timestamp, and stores it in a database. It also runs server-side alert checks ‚Äî if moisture drops critically, it triggers an SMS/email notification immediately.

Database: MongoDB / Firebase
üåê

Step 5: Website Displays Live Data

The website fetches the latest readings from the cloud API every 5 seconds using JavaScript. It renders color-coded sensor cards, pH scales, NPK badges, and Chart.js graphs ‚Äî all automatically updating without refreshing the page.

Step 6: Farmer Takes Action

The farmer opens the website on any phone, tablet, or computer. They see instant visual indicators ‚Äî green for healthy, yellow for warning, red for critical. AI recommendations tell them exactly what to do: "Water your field", "Add lime", or "All good ‚Äî no action needed!"
